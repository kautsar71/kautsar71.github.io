FORM EDIT PADA LARAVEL

UPDATE DATA MENGGUNAKAN LARAVEL

Sebelumnya saya sudah membuat view untuk menampilkan data cosheet.

Selanjutnya kita buat button edit atau link seperti berikut.



Pada tombol edit tersebut kita memerintahkan untuk mengalihkan halaman ke route "cosheet/detail" sambil mengirimkan data id, data yang ingin di edit.

Maka sekarang kita buat route baru untuk edit agar data id yang dikirimkan sekalian kita kirim juga ke controller untuk kita ambil data cosheet yang ber id tersebut.



setelah itu lalu kita buat method dalam controller CosheetController.php



Selanjutnya kita buat view dengan nama detail.blade.php karena kita akan menampilkan data cosheet yang ingin di edit.



Dan jangan lupa kita membuat route update agar data yg sudah kita ubah bisa kita simpan oleh karena itu kita buat route cosheet/update



Para route update ini kita tidak menggunakan method get, tapi menggunakan method post pada route ini kita perintahkan untuk menjalankan method update dalam controller CosheetController.php


Lalu kita coba dengan menjalankan xampp dan masuk url localhost:8000/"nama-project", Sekian untuk penjelasan kali ini, walaupun penjelasannya kurang dapat di pahami saya mohon maaf sebesar besarnya
