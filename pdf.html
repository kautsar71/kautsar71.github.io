LAPORAN PDF DENGAN DOMPDF LARAVEL

Laporan PDF dengan Dompdf Laravel


PDF masih menjadi semacam fitur yang paling dibutuhkan pada sebuah aplikasi atau sistem informasi. dengan tujuan untuk membuat cetak data, atau export data menjadi laporan yang membentuk PDF.


Salah satu cara untuk membuat cetak laporan PDF pada laravel, kita bisa menggunakan library DOMPDF. mungkin sebelumnya teman-teman sudah pernah menggunakan dan familiar dengan DOMPDF. pada saat membuat laporan PDF atau cetak laporan dengan PDF. dulu mungkin pada saat pertama kali belajar PHP dasar.


DOMPDF adalah library PHP yang bisa digunakan untuk membuat laporan PDF dengan PHP. DOMPDF bekerja dengan cara mengubah halam menjadi file PDF.


DOMPDF juga tersedia dan bisa kita gunakan pada laravel. dan cara menggunakan DOMPDF Laravel pun sangat mudah.


PERSIAPAN DATA

sebelum membuat laporan kita akan membutuhkan sebuah data, dimana data tersebut yang nantinya akan di panggil dalam file pdf


selanjutnya buat route baru untuk menampilkan data contoh(cosheet) dari database dan untuk menampilkan data (cosheet) sebagai pdf.


Route yang pertama '/cosheet' akan mengakses method index() dalam controller CosheetController.php route kedua '/cosheet/cetak_pdf' kata perintah untuk mengakses method cetak_pdf() dalam controller CosheetController.php


karena kita belum punya controller CosheetController.php maka sekarang buat controller tersebut. Saya membuat controller CosheetController.php


setelah selesai membuat controller tersebut, silahkan buka file tersebut


dengan loadview() kita menggunakan dompdf untuk membuka viewnya, kita isi nama view untuk halaman cetak pdf ini dengan view printcosheet.blade.php sambil kita passing data cosheet juga ke view ini


kemudian bagian terakhir kita return fungsi download() dari package dompdf untuk mencetak halaman menjadi file PDF dan mendownloadnya secara otomatis.


Penjelasan pada controller selesai. namun kita belum membuat view detail.blade.php dan printcosheet.blade.phpnya.


pada kedua buah view ini kita hanya menampilkan cosheetnya seperti biasa.


buat view baru, yaitu view detail.blade.php. dan satu lagi view printcosheet.blade.php


resources/view/detail.blade.php



resources/view/printCosheet.blade.php



oke mari kita lihat hasil contoh membuat laporan pdf dengan DOMPDF laravel ini

php artisan serve


dan akses route cosheet.
